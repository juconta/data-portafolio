\# Análisis de Consumo de Energía



Proyecto de análisis de datos utilizando Python.



\## Objetivo



Analizar el consumo eléctrico de clientes para identificar patrones de consumo por localidad.



\## Tecnologías utilizadas



\- Python

\- Pandas

\- Matplotlib



\## Análisis realizados



\- cálculo de consumo promedio

\- análisis de consumo por localidad

\- visualización de datos
