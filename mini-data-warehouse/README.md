\# Mini Data Warehouse con ETL



Proyecto que simula un pequeño Data Warehouse utilizando Python y SQLite.



\## Arquitectura



Datos crudos → proceso ETL → Data Warehouse



\## Tecnologías



\- Python

\- Pandas

\- SQLite



\## Proceso



1\. extracción de datos desde CSV

2\. transformación de datos

3\. carga en base de datos
