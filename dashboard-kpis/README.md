\# Dashboard de KPIs Empresariales



Aplicación web para visualizar indicadores de ventas utilizando Streamlit.



\## KPIs



\- Ventas totales

\- Clientes activos

\- Ventas por ciudad

\- Ventas por fecha



\## Tecnologías



\- Python

\- Pandas

\- Streamlit



\## Ejecutar el proyecto



pip install -r requirements.txt



python -m streamlit run dashboard\_kpis.py
