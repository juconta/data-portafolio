\# Web Scraping de Productos



Proyecto de extracción de datos desde una página web utilizando Python.



\## Objetivo



Recolectar información de productos (título y precio) desde un sitio web y almacenarla en un dataset para análisis posterior.



\## Tecnologías



\- Python

\- requests

\- BeautifulSoup

\- pandas



\## Ejecución



Instalar dependencias:



pip install -r requirements.txt



Ejecutar scraping:



python scraping\_precios.py
