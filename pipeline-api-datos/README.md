\# Pipeline de Datos desde API



Proyecto de extracción automática de datos climáticos desde una API.



\## Objetivo



Construir un pipeline simple que obtenga datos desde una API, los procese y los almacene en un dataset.



\## Tecnologías



\- Python

\- requests

\- pandas



\## API utilizada



Open-Meteo



\## Ejecución



pip install -r requirements.txt



python api\_pipeline.py
