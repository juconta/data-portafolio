\# Customer Analytics



Proyecto de análisis de comportamiento de clientes utilizando Python.



\## Objetivo



Analizar datos de clientes para entender patrones de compra y comportamiento.



\## Tecnologías



\- Python

\- Pandas

\- Matplotlib



\## Análisis realizados



\- estadísticas generales

\- compras promedio por ciudad

\- visualización de compras por cliente
