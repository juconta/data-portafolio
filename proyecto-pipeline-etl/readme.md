\# Pipeline ETL de Datos



Proyecto de ingeniería de datos utilizando Python.



\## Objetivo



Construir un pipeline ETL que permita:



\- extraer datos desde un archivo CSV

\- limpiar registros incompletos

\- transformar los datos

\- guardar los datos procesados



\## Tecnologías utilizadas



\- Python

\- Pandas



\## Proceso ETL



1\. Extract → lectura de datos desde CSV

2\. Transform → limpieza de datos y creación de nuevas variables

3\. Load → almacenamiento de datos procesados
