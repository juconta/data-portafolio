\# NLP para Atención al Cliente



Proyecto de clasificación automática de mensajes de clientes utilizando procesamiento de lenguaje natural.



\## Objetivo



Clasificar mensajes de clientes en categorías:



\- facturación

\- problema técnico

\- consulta



\## Tecnologías



\- Python

\- Pandas

\- Scikit-learn



\## Modelo



Naive Bayes



\## Ejecución



pip install -r requirements.txt



python nlp\_clientes.py
